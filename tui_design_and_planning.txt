Waste TUI planning

the actual UI requirements and whether FTXUI can deliver.                                    
                                                                                                                                     
  Required Views Analysis                                                                                                            
                                                                                                                                     
  Based on the Windows GUI, WASTE needs these core views:                                                                            
                                                                                                                                     
  1. Network/Connections View                                                                                                        
                                                                                                                                     
  Purpose: Monitor and manage peer connections                                                                                       
  Needs:                                                                                                                             
  - List of connections with multiple columns (IP, status, nick, files shared)                                                       
  - Status indicators (connecting/authenticated/failed)                                                                              
  - Actions: add connection, disconnect, view details                                                                                
  - Real-time status updates                                                                                                         
                                                                                                                                     
  2. Search View                                                                                                                     
                                                                                                                                     
  Purpose: Find files across the network                                                                                             
  Needs:                                                                                                                             
  - Search input field                                                                                                               
  - Results table (filename, size, type, sources, user)                                                                              
  - Sortable columns                                                                                                                 
  - Select to download                                                                                                               
  - Pagination or scrolling for large results                                                                                        
                                                                                                                                     
  3. Transfers View                                                                                                                  
                                                                                                                                     
  Purpose: Monitor uploads/downloads                                                                                                 
  Needs:                                                                                                                             
  - Active transfers list                                                                                                            
  - Progress bars with percentage                                                                                                    
  - Speed (KB/s), ETA, size                                                                                                          
  - Actions: pause, resume, cancel                                                                                                   
  - Separate sections for uploads/downloads                                                                                          
                                                                                                                                     
  4. Chat View                                                                                                                       
                                                                                                                                     
  Purpose: Communicate with peers                                                                                                    
  Needs:                                                                                                                             
  - Channel/room list (sidebar)                                                                                                      
  - Message history (scrollable)                                                                                                     
  - Input field                                                                                                                      
  - Unread indicators                                                                                                                
  - Timestamps, nicknames                                                                                                            
                                                                                                                                     
  5. Browse View                                                                                                                     
                                                                                                                                     
  Purpose: Browse a specific peer's shared files                                                                                     
  Needs:                                                                                                                             
  - Tree or hierarchical folder view                                                                                                 
  - File listing                                                                                                                     
  - Select to download                                                                                                               
                                                                                                                                     
  6. Settings View                                                                                                                   
                                                                                                                                     
  Purpose: Configure the application                                                                                                 
  Needs:                                                                                                                             
  - Multiple settings categories                                                                                                     
  - Text inputs, toggles, number inputs                                                                                              
  - Save/cancel actions                                                                                                              
  - Key management (generate, import, export with password)                                                                          
                                                                                                                                     
  7. Shared Files View                                                                                                               
                                                                                                                                     
  Purpose: Manage what you're sharing                                                                                                
  Needs:                                                                                                                             
  - List of shared directories                                                                                                       
  - Add/remove directories                                                                                                           
  - File count, rescan button

  Key Shortfalls                                                                                                                     
                                                                                                                                     
  1. No table component - Search results and transfers need multi-column display. Must build custom renderer.                        
  2. No tree view - File browsing would need custom implementation.                                                                  
  3. Thread safety - Network events come from another thread. Need ScreenInteractive::PostEvent() or atomic flags with periodic      
  refresh.                                                                                                                           
  4. Scrollable text regions - Chat history needs custom scrollable container.
